# 精选商城小程序

一个功能完善的电商小程序，基于 mini-render 渲染引擎实现。

## 功能特性

### 核心功能
- 🏠 **首页** - 轮播Banner、分类导航、限时特惠、热销推荐、新品上架
- 📂 **分类页** - 左侧分类菜单、右侧商品列表、子分类展示
- 🛒 **购物车** - 商品管理、数量调整、批量选择、价格计算
- 👤 **个人中心** - 用户信息、订单管理、地址管理、功能入口
- 📦 **商品详情** - 商品展示、参数说明、用户评价、立即购买
- 📋 **订单管理** - 订单列表、状态筛选、订单操作（支付/收货/取消/删除）
- 📍 **地址管理** - 收货地址列表、默认地址设置、地址编辑
- 🔍 **商品搜索** - 关键词搜索、搜索历史、热门搜索

### 技术特点
- ✨ 精美的UI设计，现代化的渐变配色
- 📱 完整的响应式布局，适配各种屏幕尺寸
- 🎯 流畅的用户交互体验
- 💾 本地数据存储，购物车和订单持久化
- 🔄 自定义 TabBar 组件，带有购物车数量提示

## 页面结构

```
pages/
├── index/          # 首页
├── category/       # 分类页
├── cart/          # 购物车
├── profile/       # 个人中心
├── detail/        # 商品详情
├── orders/        # 订单管理
├── address/       # 地址管理
└── search/        # 商品搜索

components/
└── custom-tab-bar/ # 自定义 TabBar
```

## 如何运行

1. 确保已安装 Rust 和相关依赖
2. 在项目根目录下执行：
   ```bash
   cargo run --release --bin mini-app-window
   ```
3. 选择 `sample/real-sample` 目录作为小程序根目录

## 主要功能说明

### 购物车功能
- 支持商品添加/删除
- 支持数量调整
- 支持批量选择
- 自动计算总价
- 数据本地持久化

### 订单功能
- 支持多种订单状态（待付款、待发货、待收货、已完成）
- 支持订单筛选
- 支持订单操作（支付、取消、确认收货、删除）
- 订单状态实时更新

### 搜索功能
- 支持关键词搜索
- 搜索历史记录
- 热门搜索推荐
- 模糊匹配搜索结果

## 数据存储

小程序使用本地存储保存以下数据：
- 购物车数据
- 订单记录
- 搜索历史

## 样式规范

### 颜色规范
- 主色调: `#FF6B35` (橙红色)
- 辅助色: `#4A90D9` (蓝色)
- 成功色: `#52C41A` (绿色)
- 警告色: `#FA541C` (橙色)
- 背景色: `#F5F5F5` (浅灰)

### 圆角规范
- 卡片圆角: `16rpx`
- 按钮圆角: `24rpx~40rpx`
- 图片圆角: `8rpx~12rpx`

### 间距规范
- 页面边距: `20rpx`
- 卡片内边距: `24rpx`
- 元素间距: `12rpx~20rpx`
