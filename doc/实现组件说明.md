# 微信小程序内置组件完整列表（2025年12月）

根据微信小程序官方文档（https://developers.weixin.qq.com/miniprogram/dev/component/），截至当前日期（2025年12月28日），微信小程序共提供 **68 个** 内置组件（不包括自定义组件或插件组件）。

以下按官方分类整理成 Markdown 格式，包含每个组件的名称，并附带简要说明（部分组件有典型使用场景示例）：

### 视图容器（View Container） - 11 个
- `view`：最基础的视图容器，支持 flex 布局
- `scroll-view`：可滚动的视图容器，常用于长列表
- `swiper`：轮播图组件
- `swiper-item`：轮播图的子项
- `cover-view`：覆盖在原生组件之上的视图（常用于地图、视频上叠加文字/按钮）
- `cover-image`：覆盖在原生组件之上的图片
- `movable-area`：可移动区域容器
- `movable-view`：可移动的视图（常用于拖拽元素）
- `match-media`：媒体查询组件
- `page-container`：页面容器（用于页面过渡）
- `root-portal`：根节点门户（用于全局浮层）

### 基础内容（Basic Content） - 5 个
- `text`：文本，支持长按选中
- `icon`：图标组件（内置多种图标类型）
- `progress`：进度条
- `rich-text`：富文本，支持 HTML 标签
- `selection`：文本选择组件

### 表单组件（Form Components） - 17 个
- `form`：表单容器
- `button`：按钮，支持多种类型（submit、loading 等）
- `checkbox`：复选框
- `checkbox-group`：复选框组
- `radio`：单选框
- `radio-group`：单选框组
- `input`：输入框
- `textarea`：多行文本域
- `slider`：滑动选择器
- `switch`：开关选择器
- `picker`：选择器（日期、地区、时间等）
- `picker-view`：嵌入式选择器
- `picker-view-column`：选择器列
- `label`：表单标签
- `editor`：富文本编辑器
- `editor-portal`：编辑器门户
- `keyboard-accessory`：键盘附件（自定义键盘工具栏）

### Skyline 渲染引擎组件 - 23 个
Skyline 是高性能渲染框架，包含手势和布局组件：
- `tap-gesture-handler`：点击手势
- `double-tap-gesture-handler`：双击手势
- `long-press-gesture-handler`：长按手势
- `pan-gesture-handler`：拖动手势
- `horizontal-drag-gesture-handler`：水平拖动手势
- `vertical-drag-gesture-handler`：垂直拖动手势
- `scale-gesture-handler`：缩放手势
- `force-press-gesture-handler`：压力手势
- `draggable-sheet`：可拖拽的底部弹层
- `grid-view`：网格视图
- `grid-builder`：网格构建器（配合 custom scroll-view）
- `list-view`：列表视图
- `list-builder`：列表构建器
- `nested-scroll-body`：嵌套滚动主体
- `nested-scroll-header`：嵌套滚动头部
- `sticky-header`：粘性头部
- `sticky-section`：粘性分区
- `span`：行内布局容器
- `snapshot`：快照组件
- `share-element`：共享元素过渡
- `open-container`：打开容器
- `open-data-item`：开放数据项
- `open-data-list`：开放数据列表

### 导航（Navigation） - 2 个
- `navigator`：页面跳转导航
- `functional-page-navigator`：功能页导航（插件专用）

### 媒体组件（Media Components） - 9 个
- `image`：图片，支持懒加载
- `video`：视频播放器
- `audio`：音频播放器（基础版）
- `live-player`：直播播放器
- `live-pusher`：直播推流
- `camera`：摄像头组件

### 地图（Map） - 1 个
- `map`：地图组件（支持标记、控件等）

### 画布（Canvas） - 1 个
- `canvas`：画布组件（2D/3D）

### 开放能力（Open Capabilities） - 10 个
- `web-view`：嵌入网页
- `ad`：原生广告
- `ad-custom`：自定义广告
- `official-account`：公众号关注组件
- `official-account-publish`：公众号发布组件
- `open-data`：开放数据展示
- `store-coupon`：小商店优惠券
- `store-gift`：小商店礼品
- `store-home`：小商店首页
- `store-product`：小商店商品

### 其他特殊组件 - 5 个
- `page-meta`：页面元信息配置（自定义导航栏颜色等）
- `navigation-bar`：自定义导航栏
- `native-component`：原生组件说明
- `aria-component`：无障碍访问组件
- `root-portal`（已在上方视图容器中列出，但有时单独提及）

**总计**：68 个组件

**备注**：
- Skyline 组件主要用于新渲染引擎（高性能场景），需在 `app.json` 中启用。
- 部分组件（如 `cover-view`、`cover-image`）在同层渲染普及后使用场景减少，但仍为官方内置。
- 微信小程序持续更新，建议随时查阅官方文档以获取最新列表和变更。
